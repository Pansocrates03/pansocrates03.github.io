WP Report Manager

Project made for Whirpool’s Latam Tech division.

Abstract

Whirpool’s promotor report manager is the system that makes report-creation easier. Users will be able to create a report of any unconformity from their mobile phone (ios/andriod). This report will obtain main information, such as the location, the reason, the level of urgency, and images.

Once the report is done, it will be sent to a database that an administrator will be able to access it from a web application where the admin will be able to evaluate the report and assign points to the user if needed. The user then will be able to buy stuff with those points.
